# AnachronDB 🕰️

**AnachronDB** is an immutable, time-traveling, high-speed in-memory database for developers who need full historical insights with transactional safety.

> 🚧 **Alpha version (`v0.1.0-alpha`) - highly experimental and under heavy development.**

## Features

- ✅ Immutable event-sourced storage.
- ✅ Time-travel queries.
- ✅ Version history for keys.
- ✅ Diffing between key versions.
- ❌ Persistence (coming soon).
- ❌ Networking (coming soon).
